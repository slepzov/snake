Учебный проект - игра "Змейка"

Использовался Python 3.10.2 и библиотека Pygame 2.1.2.

Запуск игры с помощью команды в cli: python main.py

Управление змейкой осуществляется клавишами 'W', 'S', 'A', 'D'.
